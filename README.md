# 武汉肺炎疫情实时动态省市地图

> 武汉加油，众志成城，共抗疫情

最近所有人的注意力都集中在武汉疫情上，在 [丁香园肺炎疫情实时动态](https://3g.dxy.cn/newh5/view/pneumonia?from=timeline) 中提供了疫情地图及实时权威新闻，但其中并没有省市地图。

根据[原仓库](https://github.com/shfshanyue/2019-ncov)前端使用vue重构，且调整了数据源，最后提醒大家尽量不出门，出门后必戴口罩。
